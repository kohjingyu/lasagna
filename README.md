# Lasagna

## Directory Structure
All files can be downloaded from the [im2recipe site](http://im2recipe.csail.mit.edu/dataset/download/). Our code expects a directory structure as follows:

```
data/
    lmdbs/
        train_keys.pkl
        train_lmdb
        val_keys.pkl
        val_lmdb
        test_keys.pkl
        test_lmdb
    images/
        train/
        val/
        test/
```
